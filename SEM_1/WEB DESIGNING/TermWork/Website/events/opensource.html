Open Source

Open source is a term that refers to anything that is available to the public for use and/or modification from its original design. Open source software is a type of software that is publicly accessible and allows users to modify the code. Open source hardware is a type of hardware that is publicly accessible and allows users to modify the design. Open source is a philosophy that promotes the freedom to use, study, share, and modify software and hardware. Open source is a movement that encourages the sharing of knowledge and skills to benefit the general public.
